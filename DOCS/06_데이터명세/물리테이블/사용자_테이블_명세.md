# app_users 테이블 명세

| 컬럼명 | 타입 | 제약 | 설명 |
| --- | --- | --- | --- |
| id | TEXT | PK | 사용자 ID |
| school_id | TEXT | FK, NOT NULL | 학교 tenant |
| classroom_id | TEXT | FK, NULL | 학급 |
| login_id | TEXT | UNIQUE, NOT NULL | 교직원/운영자 로그인 ID 또는 학생 내부 계정 식별자 |
| password | TEXT | NOT NULL | 교직원/운영자 비밀번호 해시, 학생은 placeholder 비밀번호 해시 |
| display_name | TEXT | NOT NULL | 표시 이름(학생 실명 포함) |
| role | TEXT | NOT NULL | `TEACHER` / `STUDENT` / `OPERATOR` |
| pin | TEXT | NULL | 학생 PIN 해시 (교직원/운영자는 null) |
| active | BOOLEAN | NOT NULL | 계정 활성 여부 |
| created_at | TIMESTAMP | NOT NULL | 생성 시각 |
