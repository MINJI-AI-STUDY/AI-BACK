# 법적 고지 및 동의 정책

## 목적
- 실제 운영 가능한 최소 고지/동의 항목을 정의합니다.
- 본 문서는 구현 가이드이며 법률 자문을 대체하지 않습니다.

## Must-have
1. 서비스 이용약관
2. 개인정보 수집·이용 동의
3. 학생 서비스 고지
4. 학교 운영자 승인/반려 로그 보관 안내

## 가입 화면 필수 표시 항목
- 수집 목적
- 수집 항목
- 보유 기간
- 동의 거부권 및 불이익
- 학교 운영자 승인 필요 여부

## 기록해야 하는 항목
- 동의 버전
- 동의 시각
- 가입 요청 역할(교사/학생)
- 학교 ID
- 승인/반려 시각 및 검토자
