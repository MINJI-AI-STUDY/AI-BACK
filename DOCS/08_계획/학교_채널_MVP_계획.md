# 학교 채널 MVP 계획

## 범위
- 학교 > 채널 구조만 사용
- PDF는 정확히 1개 채널에 연결
- 학생은 채널 워크스페이스에서 해당 채널 PDF만 사용
- 실시간은 입장자 목록과 채널 메시지만 제공
- 이미지는 채널 메시지가 아니라 챗봇 입력에만 둠

## 수동 QA 기준
1. 교사가 채널 생성 가능
2. 교사가 채널 PDF 업로드 가능
3. 학생이 같은 채널 워크스페이스에서 PDF 확인 가능
4. 학생 입장 시 교사 화면 roster에 이름이 나타남
5. 학생 메시지가 teacher workspace 최근 메시지에 반영됨
