# F4 학생 문제 풀이 및 자동 채점

## API 명세
- `GET /api/student/question-sets/{distributionCode}`
- `POST /api/student/question-sets/{distributionCode}/submissions`
- `GET /api/student/submissions/{submissionId}/result`

## 정책

### POL-SUB-001 배포 코드 진입 정책
- 코드: `POL-SUB-001`
- 내용: 학생은 유효한 배포 코드로만 문제 세트에 입장할 수 있습니다.

### POL-SUB-002 제출 횟수 정책
- 코드: `POL-SUB-002`
- 내용: 학생은 문제 세트별 최종 제출 1회만 가능합니다.

### POL-SUB-003 응답 필수 정책
- 코드: `POL-SUB-003`
- 내용: 제출 시 제외되지 않은 모든 문항에 응답해야 합니다.

### POL-SUB-004 마감 정책
- 코드: `POL-SUB-004`
- 내용: `dueAt`이 설정된 세트는 마감 이후 제출을 차단합니다.

### POL-SUB-005 자동 채점 정책
- 코드: `POL-SUB-005`
- 내용: 제출 즉시 점수와 문항별 결과를 계산해 저장합니다.

### POL-SUB-006 결과 노출 정책
- 코드: `POL-SUB-006`
- 내용: 학생 결과와 해설은 제출 완료 후에만 조회할 수 있습니다.

### POL-SUB-007 예외 처리 정책
- 코드: `POL-SUB-007`
- 내용: 잘못된 배포 코드, 중복 제출, 마감된 세트, 누락 응답은 모두 제출 실패로 처리합니다.
