# F5 결과 확인 및 대시보드

## API 명세
- `GET /api/student/submissions/{submissionId}/result`
- `GET /api/teacher/question-sets/{questionSetId}/dashboard`
- `GET /api/operator/overview`

## 정책

### POL-DASH-001 학생 결과 노출 정책
- 코드: `POL-DASH-001`
- 내용: 학생은 본인이 제출한 결과와 해설만 조회할 수 있습니다.

### POL-DASH-002 교사 조회 범위 정책
- 코드: `POL-DASH-002`
- 내용: 교사는 자신이 배포한 문제 세트의 학생별 점수, 문항별 정답률, 취약개념만 조회할 수 있습니다.

### POL-DASH-003 운영자 조회 범위 정책
- 코드: `POL-DASH-003`
- 내용: 운영자는 학생 상세 없이 평균 점수, 참여율, 완료율만 조회할 수 있습니다.

### POL-DASH-004 집계 시점 정책
- 코드: `POL-DASH-004`
- 내용: MVP에서는 조회 시점에 실시간으로 집계합니다.

### POL-DASH-005 취약개념 계산 정책
- 코드: `POL-DASH-005`
- 내용: 취약개념은 오답 문항의 concept tag count 내림차순으로 계산합니다.

### POL-DASH-006 예외 처리 정책
- 코드: `POL-DASH-006`
- 내용: 제출 데이터가 없으면 0 또는 빈 목록으로 응답하고, 권한 없는 조회는 차단합니다.
