# 로컬 실행 가이드

## 포트
- Frontend: `5173`
- API: `8080`
- AI: `8000`

## 실행 순서
1. AI 서버 실행
2. API 서버 실행
3. Frontend 실행

## 실행 명령

### AI
```powershell
cd ai
C:\Users\alswl\AppData\Local\Programs\Python\Python312\python.exe -m uvicorn app.main:app --host 0.0.0.0 --port 8000
```

### API
```powershell
cd api
./gradlew.bat bootRun --no-daemon
```

### Frontend
```powershell
cd frontend
npm install
npm run dev
```

## 접근 경로
- Frontend: `http://localhost:5173`
- API Health: `http://localhost:8080/api/health`
- AI Health: `http://localhost:8000/health`
